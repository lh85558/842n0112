## 使用方法

### 1. 安装固件
- 通过Web界面上传固件
- 或使用TFTP方式刷入

### 2. 配置打印机
1. 登录管理界面: http://192.168.10.1
2. 连接USB打印机到路由器
3. 进入 "服务" -> "打印服务"
4. 点击 "刷新" 按钮检测打印机
5. 选择打印机并点击 "添加"
6. 安装对应驱动

### 3. 客户端设置

#### Windows
1. 打开 "控制面板" -> "设备和打印机"
2. 点击 "添加打印机"
3. 选择 "网络打印机"
4. 输入: `http://192.168.10.1:631/printers/打印机名称`
5. 安装对应驱动

#### macOS
1. 打开 "系统偏好设置" -> "打印机与扫描仪"
2. 点击 "+" 添加打印机
3. 选择 "IP" 标签页
4. 地址: `192.168.10.1`
5. 协议: `IPP`
6. 队列: `/printers/打印机名称`
7. 安装对应驱动

#### Linux
1. 打开 "设置" -> "打印机"
2. 点击 "添加打印机"
3. 选择 "网络打印机"
4. 输入: `ipp://192.168.10.1:631/printers/打印机名称`
5. 安装对应驱动

## 注意事项
- 确保USB打印机支持Linux驱动
- 建议使用2A以上的电源适配器
- 打印机需要在路由器启动前连接
- 如遇问题可尝试重启路由器
